Acerca de la Aplicación

Bienvenido a mi aplicación de dibujo interactivo, diseñada para ofrecer una experiencia creativa y accesible desde cualquier dispositivo. Esta aplicación ha sido desarrollada con la intención de que puedas expresar tus ideas visuales fácilmente, sin necesidad de herramientas complicadas.

Funcionalidades principales

  • Dibujo libre:  Usa el lápiz para crear trazos únicos y personalizados.

  • Formas geométricas:  Dibuja figuras predefinidas como círculos, rectángulos, y mucho más.

  • Relleno:  Añade color a las formas dibujadas con un solo clic.

  • Texto:  Inserta y personaliza texto dentro del lienzo, eligiendo fuentes, tamaños y estilos.

  • Borrador:  Elimina partes no deseadas de tu dibujo de forma sencilla.

    Nota:  Cuando se trabaja en la versión web desde una PC, se puede aumentar o decrementar el tamaño del borrador usando las teclas + y - respectivamente.

  • Estilos de trazo y texto::  Personaliza el grosor y estilo de los trazos y el texto para crear composiciones originales.

  • Guardar tu trabajo:  Guarda tus creaciones para continuar trabajando en ellas más adelante o compartirlas con otros.

    Nota:  Cuando se trabaja en la versión web desde una PC, se puede abrir la ventana para guardar el dibujo usando la combinación de teclas Ctrl + S. Para usar esta funcionalidad debes iniciar sesión.

Tecnologías utilizadas

Esta aplicación ha sido desarrollada utilizando tecnologías modernas y robustas como Angular, Bootstrap para una interfaz responsive y amigable, y Firebase para gestionar el registro, autenticación y almacenamiento en la nube.

Sistema de registro y login

He integrado un sistema de autenticación para que puedas crear y almacenar tus dibujos de manera segura. Simplemente crea una cuenta o inicia sesión para acceder a todas las funcionalidades de la aplicación.

Propósito

Mi objetivo es brindar una herramienta intuitiva para usuarios de todas las edades y niveles de experiencia, permitiendo que la creatividad fluya sin barreras. Ya sea que desees dibujar por diversión o trabajar en un proyecto más serio, esta aplicación tiene las herramientas que necesitas.
